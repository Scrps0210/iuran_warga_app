# Aplikasi Iuran Warga (RT/RW)

Aplikasi manajemen data warga dan iuran bulanan berbasis web menggunakan Laravel dan TailAdmin.

## Fitur Utama
- **Manajemen Data Warga** (NIK, Nama, Alamat, Status)
- **Manajemen Pembayaran Iuran** (Bulan, Tahun, Status Lunas/Belum Lunas)
- **Dashboard Statistik**
- **Template Admin Premium** menggunakan TailAdmin

## Quick Start
```bash
composer run setup
composer run dev
```
